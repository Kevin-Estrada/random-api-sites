Alcohol Drink Generator

Hey Everyone! Welcome to the alcohol drink generator site. Use the bottom dropdown to either choose a random drink or if you are feeling indecisive about a certain liquor, then choose a liquor and it will generate a random drink within that category. Just to give some nerd description for those interested, this utilizes a free alcohol API that retrives all the details about the drink based on a category chosen from the drop down.

This option will return a random drink!
